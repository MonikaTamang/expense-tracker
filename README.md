# 💰 Expense Tracker (React + Vite)

A modern and responsive Expense Tracker web application built using React, Vite, and Tailwind CSS.  
It helps users track expenses, and manage their budget with visual insights using charts.

---

## 🚀 Live Demo
https://expense-tracker.vercel.app

---

## 📌 Features

- 📊 Dashboard with income & expense overview
- 💰 Add / delete income and expenses
- 🗂️ Category-based expense tracking
- 📈 Interactive charts using Recharts
- 🔍 Real-time balance calculation
- 📱 Fully responsive UI (mobile & desktop)
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router
- Recharts
- React Icons
- React Hot Toast
